
 ## ETH Art Gallery


Local website scrolls through your art.  

Set up config.json to your art urls 

https://eth-art-gallery.vercel.app/#/ 


  
## How to Run
1. install node10
2. install dependencies with 'npm install'
3. npm run dev

OR 

npm run build && npm run express
